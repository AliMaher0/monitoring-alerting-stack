DevOps Monitoring & Alerting Project

Prometheus + Grafana + Alertmanager + Docker Compose

Project Overview

This project demonstrates a full DevOps monitoring stack using Docker Compose. It includes:

  • Prometheus: Metrics collection and alerting.
  • Alertmanager: Receives alerts from Prometheus and routes notifications.
  • Grafana: Visualization dashboards for metrics.
  • Node Exporter & cAdvisor: Collect host and container metrics.
  • Sample App: Python app exposing Prometheus metrics.

Docker Compose Setup

docker-compose up -d --build
docker-compose ps
docker-compose logs -f prometheus grafana alertmanager

All services are containerized and run via Docker Compose. Prometheus, Grafana, Alertmanager, Node Exporter, cAdvisor, and sample-app are included.

Summary

This project demonstrates a modern DevOps monitoring and alerting stack, useful for monitoring applications and host systems in real-time. The stack can be extended with custom alerts, Slack/email notifications, and more complex dashboards.
